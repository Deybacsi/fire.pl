# fire.pl
A small text fire effect in perl :)

# Requirements

Perl

Modules:
- Term/Ansiscreen
- Term/Readkey
- Time/Hires

#  Installation

```
sudo cpan Term::ANSIScreen
sudo cpan Term::ReadKey
sudo cpan Time::HiRes
```

# How to use

Just run with `./fire.pl` or `./fire2.pl`
